    Exercice 1.1⚓︎

    L'énoncé⚓︎

    Écrire une fonction recherche qui prend en paramètres caractere, un caractère, et mot, une chaîne de caractères, et qui renvoie le nombre d’occurrences de caractere dans mot, c’est-à-dire le nombre de fois où caractere apparaît dans mot.

    🐍 Script Python

>>> recherche('a',"mississippi")

    Pas grand chose mis à part cette distinction caractère/chaîne de caractères qui n'existe pas en Python. La personne qui a rédigé est sûrement une habituée d'Ocaml en prépa :) Il y a aussi cette espace après la virgule des appels de recherche plus ou moins existante... Enfin le choix du nom recherche est discutable car cela ferait plus penser à un test renvoyant un booléen (je recherche i dans mississippi). On aurait pu choisr compte_lettres, nb_occurrences, compte_occurrences, etc.

Mais ici, on travaille en Python...Le code de la fonction est créé une bonne fois pour toute à la création de la fonction i.e. le paramètre par défaut est lié une bonne fois pour toute à solution. Une erreur classique mais délicate à comprendre pour les élèves et qu'on attend sûrement pas à trouver dans un sujet.

    🐍 Script Python
    def recherche_min_max(tab):
    if len(tab) == 0:
        return  {'min': None, 'max': None}
    dic = {'min': tab[0], 'max': tab[0]}
    for nombre in tab[1:]:
        if nombre < dic['min']:
            dic['min'] = nombre
        if nombre > dic['max']:
            dic['max'] = nombre
    return dic
